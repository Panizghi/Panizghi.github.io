Experience

Financial Stream Azure ML

I had a chance to work as tech project lead in my most recent co-op at Waterloo experience accelerated in collaboration with Microsoft. I worked with a group of five engineering students to work on social problems and use cloud base system to provide a solution. The stream we decided to work on was financial fraud. Within our excessive research, we learned about the huge increase in data breaches, credit card fraud scams and around 3.1 Billion dollar loss just within the last two years. The spar ignited! In this project, I was responsible for building deploying and testing a system which was able to detect scam patterns and send warning emails through the automation process. Some of the main responsibilities included:

  • Designed machine learning model to forecast fraudulent credit card transactions affecting 500,000 users
  • Contributed ideas and suggestions in team meetings and delivered updates on deadlines, designs, and enhancements by establishing clear goals, managing timelines, making proactive adjustments and strategic plans
  • Implemented customized Machine Learning Model in Azure Ml studio by constructing pipeline components from scratch to prepare the data, train and score the model
  • Prototyped the model by testing more than 56,000 transactions and visualized the result with confusion matrices to map and simplify the analysis of heavily numeric data and reports with 98% accuracy
  • Demonstrated advanced understanding of Microsoft Azure tools and infrastructure by quickly learning and applying them to the addressed problem
  • Reduced 23% on budgeting costs by utilizing Microsoft cost management and altering cost-efficient virtual machines as compute target, compute instances and clusters
  • Consulted with team mentors to develop improvement plans such as changing the algorithm model from classification to random forest.
